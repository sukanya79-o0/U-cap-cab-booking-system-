# UCab React Client Frontend

This directory contains the user interface, component controls, and state management rendering the vehicle booking ecosystem.

## Core Interface Views
- **Authentication Screens**: Secure login options tailored dynamically to separate Admin and Passenger views.
- **Booking Hub**: Interface presenting immediate pickup details, live date scheduling, and destinations.
- **Admin Dashboard**: Portal built to handle live management, addition, and configuration updates for fleet vehicles.

## Installation & Setup
1. Move to the directory: `cd frontend`
2. Install dependencies: `npm install`
3. Start the interface: `npm run dev` (or `npm start`)
